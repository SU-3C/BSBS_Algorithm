### 문제 푸는 방법

1. 시간을 정해놓고, 그 시간을 넘겨도 못 풀면 답을 본다. 처음에는 많은 풀이와 사실을 아는 게 더 중요하다.
2. 다른 사람의 코드를 본다. 좋은 코딩법을 배울 수 있고, 코드를 읽고 이해하는 능력을 키울 수 있다.
3. 코드를 작성 할 때에는 항상 명명 규칙과 인덴테이션, 주적문을 다는 것을 습관으로 한다.
4. 풀이를 논리적으로 설명할 수 있을 때 코딩을 한다.
